# Airbnb Price Factors
Analyzes Airbnb listings in New York City to understand which features most strongly influence listing prices

## Key Insights from Model Feature Importance

The Random Forest model was trained to understand what features most influence Airbnb listing prices in New York City.  

### Top Influencing Features
1. **Room Type:** Private and shared rooms tend to be priced lower than entire homes/apartments.
2. **Location:** Latitude and longitude are strong predictors, confirming that neighborhood location drives pricing.
3. **Host & Availability:** Listings with higher availability or hosted by multi-listing hosts show pricing patterns.
4. **Reviews:** Listings with more reviews or higher monthly review activity can command higher prices.
5. **Other Factors:** Minimum nights and recency of last review have minor influence.

**Takeaway:** Rather than providing a perfect price prediction for individual listings, the model highlights which features are key drivers of price. This is valuable for hosts to **optimize pricing** and for analysts to **understand market trends**.
